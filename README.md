## 🧩 Proyectos destacados

### 🧱 Naylamp — Boilerplate avanzado para SaaS

Boilerplate pensado para construir aplicaciones SaaS escalables, mantenibles y listas para extender nuevas funcionalidades.

**Características principales:**

- Arquitectura basada en **vertical slicing**
- Organización por **features y acciones de negocio**
- Ejemplo de estructura: `products/create_product`
- Separación clara por funcionalidades independientes
- Autenticación y autorización con enfoque RBAC
- Base para productos SaaS modulares
- Uso de IA para acelerar diseño, refactorización, testing y documentación

**Stack:** Go, TypeScript, PostgreSQL, RBAC, IA aplicada, agentes de desarrollo.

---

### 🤖 Proyecto RAG universitario

Sistema basado en **Retrieval-Augmented Generation**, integrando modelos de lenguaje con recuperación semántica de información.

**Características principales:**

- Búsqueda semántica usando embeddings
- Integración con **Pinecone**
- Uso de **LangChain** para chains y recuperación de contexto
- Almacenamiento vectorial con **pgvector**
- Exploración de modelos open source de **Hugging Face**
- Procesamiento de documentos, chunking y similarity search

**Stack:** Python, PostgreSQL, pgvector, Pinecone, LangChain, Hugging Face, LLMs.

---

### 🛒 Vitaca — Constructor de tiendas online con IA

Plataforma SaaS para creación de e-commerce con automatización de ventas mediante chatbot de WhatsApp.

**Características principales:**

- Creación de tiendas online
- Automatización de ventas con IA
- Integración de LLMs como GPT, Claude, Gemini y DeepSeek
- Flujos conversacionales para atención y ventas
- Uso de Whisper para speech-to-text
- Uso de TTS para respuestas por voz
- Frontend moderno con UI limpia y responsive

**Stack:** Next.js, Node.js, PostgreSQL, React Query, Zustand, TailwindCSS, Shadcn UI, LLMs, WhatsApp.

### 💬 Vento CRM — CRM de WhatsApp
